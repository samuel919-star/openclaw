# 📚 亚马逊初学者教程

一个互动式、图文结合的亚马逊学习教程，帮助初学者从零开始掌握亚马逊相关技能。

## 🌟 教程特色

- ✅ **互动式学习** - 包含小测验、计算器、检查清单
- ✅ **图文结合** - 直观的界面展示和步骤说明
- ✅ **实战导向** - 提供代码示例和操作步骤
- ✅ **进度跟踪** - 显示学习进度
- ✅ **即时反馈** - 答题和练习实时反馈

## 📖 教程内容

### 第1章：欢迎来到亚马逊世界 🚀
了解亚马逊的规模、影响力和学习价值

### 第2章：成为亚马逊卖家 🛒
掌握卖家类型、开店流程和费用计算

### 第3章：Amazon AWS 云服务 ☁️
学习AWS核心服务、免费套餐和CLI操作

### 第4章：Amazon 开发者平台 👨‍💻
了解开发者服务、账户创建和Alexa开发

### 第5章：实战技巧和最佳实践 💡
掌握成功秘诀、数据分析工具和学习资源

## 🚀 快速开始

### 方法1：使用启动脚本（推荐）

```bash
./start-amazon-tutorial.sh
```

### 方法2：直接打开文件

在浏览器中打开 `amazon-beginner-tutorial.html` 文件

### 方法3：使用本地服务器

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000/amazon-beginner-tutorial.html
```
