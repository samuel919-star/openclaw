# 📚 英语学习资源包

主人，这是为你准备的英语学习资料！

## 📖 已下载的资源

### 📝 单词列表
- **cefr_beginner.txt** - 初级词汇 (A1/A2) - 535 个单词
- **cefr_intermediate.txt** - 中级词汇 (B1/B2) - 988 个单词

### 🛠️ 学习工具
- **word_practice.py** - 单词练习助手

## 🚀 如何使用

### 方法 1: 使用单词练习脚本

```bash
python3 word_practice.py
```

脚本会提示你选择难度级别：
1. 初级 (A1/A2)
2. 中级 (B1/B2) - 适合新概念英语第二册学习者
3. 高级

然后每次随机练习一定数量的单词。

### 方法 2: 直接查看词汇列表

```bash
# 查看初级单词
head -50 cefr_beginner.txt

# 查看中级单词
head -50 cefr_intermediate.txt
```

### 方法 3: 随机抽取单词练习

```bash
# 随机抽取 10 个初级单词
sort -R cefr_beginner.txt | head -10

# 随机抽取 20 个中级单词
sort -R cefr_intermediate.txt | head -20
```

## 📋 学习建议

### 新概念英语第二册学习者（中级）

1. **每天学习 15-20 个单词**
   ```bash
   sort -R cefr_intermediate.txt | head -20
   ```

2. **学习步骤：**
   - 查阅每个单词的含义和发音
   - 造句练习
   - 记录生词本

3. **结合 B 视频课程**
   - 搜索"新概念英语第二册"
   - 每天学习 1-2 课

### 初级学习者

1. **每天学习 10-15 个单词**
   ```bash
   sort -R cefr_beginner.txt | head -15
   ```

2. **重点掌握：**
   - 基础词汇
   - 常用动词和名词
   - 日常用语

## 💡 学习技巧

1. **重复记忆**：使用艾宾浩斯遗忘曲线复习
2. **场景记忆**：将单词放入句子中记忆
3. **听说结合**：边听边跟读
4. **造句练习**：主动使用单词造句
5. **记录生词**：建立个人生词本
